## 4. Link Equity Distribution Strategy

### Priority Pages (Highest Internal Link Count Target)

These pages are the most commercially valuable and should receive the most internal links.

| Page | Target Inbound Internal Links | Rationale |
|------|-------------------------------|-----------|
| /partners | 25-30 | Primary conversion page for practice acquisition (highest value) |
| /blog/sell-dental-practice-guide (Cluster 1 Pillar) | 15-20 | Hub page for highest-value keyword cluster |
| /careers | 15-20 | Conversion page for recruitment pipeline |
| /implants | 15-20 | Conversion page for patient acquisition (MODIS vertical) |
| /why-sga | 12-15 | Comparison/education page supporting all B2B funnels |
| /specialists | 10-12 | Conversion page for specialist pipeline |
| /blog/dental-implants-guide (Cluster 3 Pillar) | 10-12 | Hub page for patient-facing content |
| /blog/dental-career-paths-guide (Cluster 2 Pillar) | 8-10 | Hub page for career content |
| /blog/evolution-of-dsos (Cluster 4 Pillar) | 8-10 | Hub page for thought leadership |

### Link Equity Flow

```
                High Authority
                    |
    Homepage -------+
    (passes equity to all top-level pages via global nav)
                    |
    +------+--------+---------+------+------+
    |      |        |         |      |      |
 /partners /careers /implants /why  /spec  /about
    |      |        |         |      |
    v      v        v         v      v
  Pillar  Pillar   Pillar   Pillar  (links to
  Pages   Pages    Pages    Pages   Cluster 3+4)
    |      |        |         |
    v      v        v         v
  Spoke   Spoke    Spoke    Spoke
  Articles Articles Articles Articles
    |      |        |         |
    +------+--------+---------+
    |
    v (cross-cluster bridges distribute equity laterally)
    |
  Location Pages (receive equity from relevant blog content + directory)
```

### Rules for Link Equity Management

1. **Every page gets at least 3 internal inbound links.** No orphan pages.
2. **Pillar pages are the equity conduits.** They receive links from all their spokes and pass equity to the landing page via CTA links.
3. **Landing pages receive links from both pillar pages and individual articles.** Every article's CTA links to the relevant landing page.
4. **Cross-cluster bridges are limited to 2-3 per article.** Too many cross-cluster links dilute topical relevance signals.
5. **Location pages receive links from the /practices directory and from relevant blog content.** Implant-focused location pages are linked from Cluster 3 articles.
6. **No deep pages should be more than 3 clicks from the homepage.** Crawl depth: Homepage > Section > Pillar > Spoke. Location pages: Homepage > Practices > Location.

## 5. Anchor Text Guidelines

### Rules

1. **Use descriptive, keyword-relevant anchor text.** The anchor text should accurately describe the destination page.
2. **Vary anchor text across links to the same page.** Do not use the exact same anchor text in every link to a given page. Use natural variations.
3. **Never use generic anchors as the sole link.** "Click here," "learn more," and "read more" should only appear alongside descriptive text (e.g., "Learn more about dental practice valuations").
4. **Match anchor text to the target page's primary keyword** for at least 40% of inbound links. Use secondary keywords and natural language for the remaining 60%.
5. **CTA links can use action-oriented anchor text** ("Start a confidential conversation," "Take the assessment") -- these serve conversion, not SEO.

## 7. Monitoring and Maintenance

| Metric | Tool | Cadence | Action Threshold |
|--------|------|---------|-----------------|
| Orphan pages (0 internal inbound links) | Screaming Frog or Sitebulb | Monthly | Fix any orphan page immediately -- add at least 3 internal links |
| Broken internal links | Screaming Frog | Monthly | Fix all 404 internal links within 48 hours |
| Average crawl depth | Screaming Frog | Monthly | If any page exceeds 4 clicks from homepage, add a shorter path |
| Internal link distribution | Screaming Frog/Ahrefs | Quarterly | If priority pages have fewer inbound links than target, add links from new content |
| Anchor text distribution | Ahrefs | Quarterly | If any page has >60% exact-match anchors, diversify with natural variations |
