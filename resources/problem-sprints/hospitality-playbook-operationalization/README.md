# Sprint: Hospitality Playbook Operationalization

**Problem**: SGA has developed a "hospitality playbook" concept built around a culture of 5-star patient experiences — but cannot operationalize it at scale. They cannot train staff on hospitality behaviors, measure adoption, or reinforce the culture systematically across a distributed network of 260+ practices.

**Sprint Slug**: `hospitality-playbook-operationalization`
**Started**: 2026-03-10
**Completed**: 2026-03-10
**Status**: Complete

---

## Solution: The Hospitality Playbook Platform

A unified, distributable operating system for "Caring with Charisma" — combining five integrated components into one flywheel that trains staff, measures behavior, rewards performance, and sustains culture across 260+ practices.

**Core concept**: The Morning Huddle is the daily ritual container. The CCA Academy creates the motivation architecture. The H-Score Engine makes culture visible and competitive. The Trust Transfer OS provides behavioral precision. The Champion Network creates human durability. All five feed each other.

**North star**: Dental schools adopt "Caring with Charisma" as curriculum — making SGA-trained staff the recognized gold standard for patient experience in dentistry.

## Platform Architecture Summary

### The Five Pillars

| Pillar | Component | Primary Output |
|--------|-----------|---------------|
| 1 | Morning Huddle OS | Daily 5-minute ritual inside Dental Intel |
| 2 | CCA Academy | Bronze / Silver / Gold certification + portable credential |
| 3 | Hospitality Score Engine | Daily H-Score per practice (Cinnamon 35% + Swell 45% + Dental Intel 20%) |
| 4 | Trust Transfer OS | Clinical handoff verification + review sentiment correlation |
| 5 | Champion Network | 260 certified Champions + peer community + wow story network |

### The Ten Platform Components

1. Playbook Content Engine — Living content library with behavior taxonomy
2. Morning Huddle OS — Daily content delivery + Dental Intel integration
3. CCA Academy — Certification engine + tiered credential issuance
4. Hospitality Score Engine — H-Score algorithm + anomaly detection
5. Trust Transfer Verification — Checklist + voice sentiment + NLP
6. Champion Network Layer — Peer community + Regional Calls + Summit
7. Recognition and Rewards Engine — Leaderboard + bonus pool + jacket fulfillment
8. Playbook Distribution System — Version control + living document
9. Network Intelligence Dashboard — Portfolio view + intervention triggers + executive reports
10. Content Production Pipeline — Grant videos + Champion stories + academic exports

### Integration Stack

| System | Integration | H-Score Weight |
|--------|-------------|----------------|
| Swell (reviews + NPS) | API → daily review data + NLP sentiment | 45% |
| Cinnamon (phone analytics) | API → answer rate + greeting phrase detection | 35% |
| Dental Intel (morning huddle) | API → completion tracking + content delivery | 20% |
| Voice recording (common areas) | NLP → greeting/escort behavior signals | Supplementary |
| LinkedIn / Credly | Credential publication + verification portal | — |

## H-Score Levels

| Level | Range | Status | Trigger |
|-------|-------|--------|---------|
| Emerging | 0–39 | Red | Intervention protocol activated |
| Developing | 40–59 | Orange | Improvement plan assigned |
| Established | 60–74 | Yellow | On track; weakest component targeted |
| Exceptional | 75–89 | Green | Bonus pool contribution unlocked |
| Elite | 90–100 | Gold | Network recognition + case study candidate |
