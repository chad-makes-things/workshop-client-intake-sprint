## Problem Statement

SGA operates 160+ dental practice locations across three platforms (SGA legacy, Gen4, MODIS) with a two-person studio team responsible for all creative asset production, print procurement, and brand governance. Practice brand quality spans a spectrum from excellent to nearly nothing. No standardized intake, no centralized asset library, no automated design workflow, and a painful print vendor situation (Staples) make creative support a bottleneck and a source of ongoing frustration. The vision: an AI-agent-powered studio system where any practice manager can request, receive, and deploy on-brand creative in under 48 hours — through a chatbot, not an email.
