## HMW Notes — 22 Total, Grouped by Theme

### Theme A: Intake & Triage (The Request Moment)
*How do we make the creative request itself smarter — so the system knows what the practice actually needs before a human designer has to interpret it?*

| # | HMW Note |
|---|----------|
| A1 | HMW make it so a practice manager can submit a complete, actionable creative brief in under 3 minutes without knowing design terminology? |
| A2 | HMW build a "medium challenge" into every print request — asking the practice to justify print before the order is placed? |
| A3 | HMW surface the digital alternative automatically every time a print format is requested, with enough specificity that the practice can actually say yes to it? |
| A4 | HMW make the intake channel live where practice managers already are (Teams, Slack, a chatbot in their practice management system) so adoption isn't an extra step? |
| A5 | HMW capture enough context at intake (purpose, audience, timeline, format, assets needed) that the design agent can start generating without a single follow-up question? |

### Theme B: Digital Asset Foundation (The Asset Problem)
*How do we make sure the system has something to work with for every practice — before a request breaks because the assets don't exist?*

| # | HMW Note |
|---|----------|
| B1 | HMW build brand foundation packages for the 50+ underserved practices as a one-time backfill — fast, templated, and good enough to unblock the automation system? |
| B2 | HMW make Canva's brand kit the single source of truth for each practice's logo, colors, and fonts — so any agent or designer pulls from the same locked source? |
| B3 | HMW create a "brand health score" for each practice that tells the studio team which practices are ready for automated design vs. which need an asset lift first? |
| B4 | HMW automate asset ingestion — so when a new practice joins SGA, their brand kit is populated in Canva within the first week of onboarding? |
| B5 | HMW use AI to generate placeholder photography for practices that have no photos — without it looking stock or generic? |

### Theme C: Design Automation (The Production Layer)
*How do we get high-quality, on-brand output from an AI design agent without a human touching every piece?*

| # | HMW Note |
|---|----------|
| C1 | HMW build a template library in Canva that covers the 10 most-requested collateral types (business card, appointment reminder, flyer, social post, TV slide, referral card, review request card, email header, door sign, welcome packet) — so 80% of requests have a template? |
| C2 | HMW design the AI-generation workflow so the output is 90% done — and the human quality gate is a 60-second yes/no, not a redesign session? |
| C3 | HMW make brand guardrails so strict in the template system that the AI can't produce anything that's technically off-brand, even if the output is uninspired? |
| C4 | HMW route custom or edge-case requests (new service launches, event signage, campaign creative) directly to the human designer, keeping them off the automation track? |

### Theme D: Human Quality Gate (The Approval Layer)
*How do we make the review step fast, scalable, and smart — so it doesn't become the new bottleneck?*

| # | HMW Note |
|---|----------|
| D1 | HMW build a tiered review system where the AI auto-approves standard templated outputs and only flags edge cases and new template types for human review? |
| D2 | HMW make the CD's quality gate a mobile-first, one-tap approve/reject — so review doesn't require opening a design tool? |
| D3 | HMW involve the practice in the preview step in a way that's frictionless — a "does this look right for your office?" moment before final approval? |

### Theme E: Vendor Procurement (The Print Channel)
*How do we make print ordering invisible to the practice manager and invisible to the studio — while still producing quality output?*

| # | HMW Note |
|---|----------|
| E1 | HMW identify and nominate a single print vendor that practices can order from directly — with pre-configured specs for every approved template type? |
| E2 | HMW build print ordering directly into the approval workflow — so that when an asset is approved, it's one click to submit the print order with the correct file, size, and quantity? |
| E3 | HMW create a vendor audit process so the studio can monitor print quality and fulfillment without touching each order? |
